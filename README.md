# excel-vba-minecraft-snowball-menu
用excel vba 写的我的世界基岩版雪球菜单（不支持新execute）

关闭时不能点X,要点关闭按钮(新版本)

$是跳过选区 §也是跳过选区

按住shift然后按回车可以输入换行

用ChatGPT和个人理解更改的（应该会有bug）

欢迎大家提出修改意见或者增加新内容
